# タスク管理(Todo)アプリ（Google認証対応 / PWA）

## ■ 概要
ユーザーごとにデータを管理できるタスク管理アプリです。  
Googleアカウントによるログイン機能を実装し、各ユーザーが自分のデータのみを操作できるよう設計しています。  
また、PWA対応によりスマートフォンでのアプリライクな利用が可能です。

---

## ■ 主な機能

- ユーザー登録 / ログイン
- Googleアカウントによるログイン（OAuth認証）
- ログアウト機能
- CRUD機能（作成 / 参照 / 更新 / 削除）
- ユーザーごとのデータ管理（他ユーザーのデータは操作不可）
- PWA対応
  - ホーム画面追加
  - Service Workerによるキャッシュ

## ■ システム構成

- PHPでサーバーサイド処理を実装
- MySQLでユーザーデータおよび投稿データを管理
- Google OAuthで認証処理を実装
- Service Workerでキャッシュ制御

---

## ■ 認証の仕組み

Google OAuthを利用し、ログイン時にGoogleアカウント情報を取得。  
取得したユーザー情報をデータベースと紐付けることで、アプリ内ユーザーとして管理しています。

---

## ■ データ管理設計

- ユーザーごとにIDを発行
- CRUDデータにユーザーIDを紐付け
- ログインユーザーのIDと一致するデータのみ操作可能

これにより、他ユーザーのデータへアクセスできないよう制御しています。

---

## ■ PWA対応

- `manifest.json` によりホーム画面追加に対応
- `service-worker.js` によりリソースをキャッシュ
- オフライン時でも一部表示可能

## ■ 苦労した点と解決

### 認証とユーザー管理の統合
Google OAuthで取得した情報とDBのユーザーを紐付ける処理に苦労しましたが、  
ユーザーIDを基準に管理することでシンプルに実装しました。

### セッション管理
ログイン状態の維持とユーザー識別の実装で課題がありましたが、  
セッションを用いて解決しました。
